## Building InsumerAPI

On-chain verification and attestation for AI agents across 31 EVM chains.

**InsumerAPI** provides ECDSA-signed boolean attestations of on-chain state — token holdings, NFT ownership, merchant credentials — through a single REST endpoint. One API call, cryptographically verifiable, no trust dependency on the issuing server.
